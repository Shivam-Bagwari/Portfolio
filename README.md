# 🌌 Personal Portfolio

A modern developer portfolio built with **React**, **Vite**, and **Tailwind CSS v4**.

This project is being built from scratch with a focus on **clean architecture**, **reusable components**, **performance**, and **modern UI/UX**. Instead of relying on heavy animation libraries, most interactions are implemented using **pure CSS** and **native browser APIs**.

> This repository is also part of my learning journey, where I'm building every feature from the ground up while following industry-level development practices.

---

## ✨ Features

- 🎨 Modern dark glassmorphism UI
- 🟣 Blueprint-inspired background
- 🌌 Aurora gradient effects
- 🖱️ Custom animated cursor
- 📈 Scroll progress indicator
- 🎯 Interactive project cards with 3D tilt
- 📊 Animated statistics
- 🟩 GitHub contribution heatmap
- ⌨️ Command Palette (⌘K / Ctrl+K)
- 📱 Fully responsive layout
- ⚡ Built with performance-first principles

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/yourusername/portfolio.git
```

Navigate into the project

```bash
cd portfolio
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview the production build

```bash
npm run preview
```
